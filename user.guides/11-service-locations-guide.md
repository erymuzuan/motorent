# Service Locations Guide

Service Locations are specific points where vehicles can be picked up or dropped off. These can be your own shops, hotels, airports, or partner locations.

## Configuration

1. Navigate to **Settings > Service Locations**.
2. Click **Add Location**.
3. Fill in the details:
   - **Name**: e.g., "Phuket Airport", "Patong Main Office".
   - **Type**: Shop, Hotel, Airport, or Other.
   - **Coordinates**: (Optional) For mapping features.
   - **Drop-off Fee**: The additional charge if a customer returns a vehicle here instead of the pickup location.
4. Click **Save**.

## Using Service Locations

### At Check-In
When processing a rental, you can specify:
- **Pickup Location**: Where the customer is taking the bike.
- **Return Location**: Where they plan to return it.

If the Return Location is different from the Pickup Location, the **Drop-off Fee** will be automatically added to the rental total.

### At Check-Out
If a customer returns a vehicle to a different location than originally planned, the staff can update the return location in the check-out dialog, which will adjust the final balance.

## Managing Fleet across Locations
The **Fleet Dashboard** shows the current location of every vehicle, making it easy to see where your inventory is distributed.
