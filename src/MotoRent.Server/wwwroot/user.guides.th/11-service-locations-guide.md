# คู่มือสถานที่ให้บริการ

สถานที่ให้บริการคือจุดเฉพาะที่สามารถรับหรือส่งคืนยานพาหนะได้ ซึ่งอาจเป็นร้านค้าของคุณเอง โรงแรม สนามบิน หรือสถานที่พันธมิตร

## การกำหนดค่า

1. ไปที่ **การตั้งค่า > สถานที่ให้บริการ**
2. คลิก **เพิ่มสถานที่**
3. กรอกรายละเอียด:
   - **ชื่อ**: เช่น "สนามบินภูเก็ต", "สำนักงานใหญ่ป่าตอง"
   - **ประเภท**: ร้านค้า, โรงแรม, สนามบิน หรืออื่นๆ
   - **พิกัด**: (ไม่บังคับ) สำหรับฟีเจอร์แผนที่
   - **ค่าธรรมเนียมคืนรถ**: ค่าใช้จ่ายเพิ่มเติมหากลูกค้าคืนยานพาหนะที่นี่แทนสถานที่รับรถ
4. คลิก **บันทึก**

## การใช้งานสถานที่ให้บริการ

### ที่เช็คอิน
เมื่อประมวลผลการเช่า คุณสามารถระบุ:
- **สถานที่รับรถ**: ที่ที่ลูกค้าจะรับรถ
- **สถานที่คืนรถ**: ที่ที่พวกเขาวางแผนจะคืนรถ

หากสถานที่คืนรถแตกต่างจากสถานที่รับรถ **ค่าธรรมเนียมคืนรถ** จะถูกเพิ่มเข้าไปในยอดรวมการเช่าโดยอัตโนมัติ

### ที่เช็คเอาท์
หากลูกค้าคืนยานพาหนะที่สถานที่อื่นนอกเหนือจากที่วางแผนไว้เดิม พนักงานสามารถอัปเดตสถานที่คืนรถในหน้าต่างเช็คเอาท์ ซึ่งจะปรับยอดคงเหลือสุดท้าย

## การจัดการฟลีตข้ามสถานที่
**แดชบอร์ดฟลีต** แสดงตำแหน่งปัจจุบันของยานพาหนะทุกคัน ทำให้ง่ายต่อการดูว่าสินค้าคงคลังของคุณกระจายอยู่ที่ไหนบ้าง
