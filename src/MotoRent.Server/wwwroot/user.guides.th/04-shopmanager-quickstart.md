# คู่มือเริ่มต้นใช้งาน MotoRent - ผู้จัดการร้าน

ยินดีต้อนรับสู่ MotoRent! คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งานในฐานะผู้จัดการร้าน

![แดชบอร์ด](images/02-dashboard.png)

## บทบาทของคุณ

ในฐานะผู้จัดการร้าน คุณมี **สิทธิ์เข้าถึงเต็มรูปแบบ** ในการจัดการการดำเนินงานของร้าน:
- ดูแลการเช่าและกิจกรรมของพนักงาน
- จัดการยานพาหนะและสินค้าคงคลัง
- ติดตามการเงินและการชำระเงิน
- จัดการอุบัติเหตุและเหตุการณ์
- กำหนดค่าการตั้งค่าร้าน

## ภาพรวมการนำทาง

เมนูนำทางของคุณประกอบด้วย:

| เมนู | คำอธิบาย |
|------|-------------|
| **แดชบอร์ด** | ภาพรวมร้านและการดำเนินการด่วน |
| **การเช่า** | การเช่าที่ใช้งานอยู่และเช็คอิน/เช็คเอาท์ |
| **ตัวแทน** | จัดการการจองพันธมิตรและค่าคอมมิชชั่น |
| **การเงิน** | การชำระเงิน เงินมัดจำ การจ่ายเงินเจ้าของ รายงาน |
| **ยานพาหนะ** | สินค้าคงคลังยานพาหนะและการบำรุงรักษา |
| **อุบัติเหตุ** | การติดตามและแก้ไขเหตุการณ์ |
| **ลูกค้า** | ฐานข้อมูลผู้เช่า |
| **การตั้งค่า** | การกำหนดค่าร้าน ประกันภัย กลุ่มยานพาหนะ |

## ความรับผิดชอบหลัก

### 1. การดำเนินงานประจำวัน
- ตรวจสอบแดชบอร์ดสำหรับการรับและคืนรถวันนี้
- ตรวจสอบให้แน่ใจว่าพนักงานเปิดและปิดลิ้นชักเงินสดอย่างถูกต้อง
- ตรวจสอบและอนุมัติเอกสารเช็คอิน

### 2. การจัดการตัวแทน
- ลงทะเบียนพันธมิตรท้องถิ่นใหม่ (โรงแรม บริษัทท่องเที่ยว)
- ติดตามการจองที่แนะนำโดยตัวแทน
- ดำเนินการจ่ายค่าคอมมิชชั่นผ่านลิ้นชักเงินสด
- ดู [คู่มือการจัดการตัวแทน](10-agent-management-guide.md)

### 3. สถานที่ให้บริการ
- กำหนดค่าจุดรับและส่งสำหรับร้านของคุณ
- ตั้งค่าธรรมเนียมการส่งสำหรับการเช่าแบบทางเดียว
- ดู [คู่มือสถานที่ให้บริการ](11-service-locations-guide.md)

### 4. ยานพาหนะและการบำรุงรักษา
- ตรวจสอบสถานะยานพาหนะ (พร้อมใช้งาน เช่าอยู่ ซ่อมบำรุง)
- กำหนดเวลาการบริการตามปกติ
- จัดการ **กลุ่มยานพาหนะ** (แชร์สินค้าคงคลังระหว่างร้านของคุณ)

#### กลุ่มยานพาหนะ
1. ไปที่ **การตั้งค่า > กลุ่มยานพาหนะ**
2. สร้างหรือเข้าร่วมกลุ่มกับร้านอื่นในองค์กรของคุณ
3. ซึ่งช่วยให้คุณให้เช่ายานพาหนะที่เป็นของร้านอื่นได้หากสินค้าคงคลังของคุณต่ำ

### 5. การเงินและราคา
- ตรวจสอบรายได้ประจำวันและวิธีการชำระเงิน
- ดำเนินการ **จ่ายเงินเจ้าของ** สำหรับยานพาหนะที่จัดการ
- จัดการ **กฎราคา** เพื่อให้แข่งขันได้

#### กฎราคา
จัดการการกำหนดราคาแบบไดนามิกสำหรับร้านของคุณภายใต้ **การตั้งค่า > กฎราคา**:
- ใช้ **ปฏิทินราคา** เพื่อแสดงอัตราที่ใช้งานอยู่
- สร้างค่าธรรมเนียมเพิ่มเติมสำหรับวันหยุดหรือวันหยุดสุดสัปดาห์
- ตั้งส่วนลดสำหรับการเช่าระยะยาว

## เริ่มต้นใช้งาน

### 1. แดชบอร์ด - ศูนย์บัญชาการของคุณ

ตรวจสอบร้านของคุณได้อย่างรวดเร็ว:
- **การเช่าที่ใช้งานอยู่** - ยานพาหนะที่ให้เช่าอยู่ในปัจจุบัน
- **ยานพาหนะที่พร้อมใช้งาน** - พร้อมให้เช่า
- **ครบกำหนดวันนี้** - การคืนที่คาดหวัง
- **รายได้วันนี้** - การเก็บเงินประจำวัน

ตาราง **การเช่าล่าสุด** แสดงกิจกรรมล่าสุด

### 2. การจัดการทีมของคุณ

#### ความรับผิดชอบของพนักงาน
พนักงานสามารถจัดการ:
- เช็คอินและเช็คเอาท์
- การลงทะเบียนผู้เช่า
- การรายงานอุบัติเหตุ

#### สิ่งที่พนักงานไม่สามารถทำได้
- เข้าถึงส่วนการเงิน
- แก้ไขราคายานพาหนะ
- เปลี่ยนการตั้งค่าร้าน

### 3. การจัดการยานพาหนะ

ไปที่ **ยานพาหนะ > ยานพาหนะทั้งหมด**

#### สินค้าคงคลังยานพาหนะ
- ดูยานพาหนะทั้งหมดตามประเภท
- ติดตามสถานะความพร้อมใช้งาน
- ตรวจสอบว่ายานพาหนะใดถูกเช่า

#### การเพิ่มยานพาหนะ
1. คลิก "+ เพิ่มยานพาหนะ"
2. กรอกรายละเอียด:
   - ป้ายทะเบียน
   - ยี่ห้อและรุ่น
   - ประเภทยานพาหนะ
   - อัตราต่อวันและเงินมัดจำ
3. ตั้งสถานะเป็นพร้อมใช้งาน

#### การจัดการสถานะยานพาหนะ
- **พร้อมใช้งาน** - พร้อมให้เช่า
- **เช่าอยู่** - กำลังให้เช่า (อัตโนมัติ)
- **ซ่อมบำรุง** - นำออกจากการให้บริการเพื่อซ่อม

### 4. ภาพรวมการเงิน

#### การชำระเงิน
ติดตามการชำระเงินค่าเช่าทั้งหมด:
- กรองตามสถานะ วิธีการ วันที่
- ดูรายละเอียดการชำระเงินตามประเภท (เงินสด บัตร โอน)
- บันทึกการชำระเงินด้วยตนเอง

#### เงินมัดจำ
ตรวจสอบเงินมัดจำค้ำประกัน:
- **ถืออยู่** - การเช่าที่ใช้งานอยู่
- **คืนแล้ว** - คืนให้ลูกค้า
- **ริบ** - เก็บไว้สำหรับความเสียหาย

#### การจ่ายเงินเจ้าของ
หากคุณจัดการยานพาหนะสำหรับเจ้าของ:
- ติดตามรายได้ของเจ้าของ
- ดำเนินการจ่ายเงินให้เจ้าของ

### 5. การจัดการอุบัติเหตุ

เมื่อพนักงานรายงานอุบัติเหตุ:
1. ตรวจสอบรายละเอียดรายงาน
2. ประเมินความรุนแรงและค่าใช้จ่าย
3. อัปเดตสถานะตามความคืบหน้า:
   - **รายงานแล้ว** - รายงานเริ่มต้น
   - **กำลังดำเนินการ** - กำลังจัดการ
   - **แก้ไขแล้ว** - เสร็จสิ้น
4. บันทึกค่าใช้จ่ายสุดท้าย

### 6. การตั้งค่าร้าน

#### แพ็กเกจประกัน
ตั้งค่าตัวเลือกประกันสำหรับลูกค้า:
1. ไปที่ **การตั้งค่า > ประกันภัย**
2. คลิก "+ เพิ่มแพ็กเกจ"
3. กำหนดค่า:
   - ชื่อแพ็กเกจ
   - อัตราต่อวัน
   - จำนวนความคุ้มครอง
   - ค่าเสียหายส่วนแรก

#### กลุ่มยานพาหนะ
แชร์ยานพาหนะข้ามร้าน:
1. ไปที่ **การตั้งค่า > กลุ่มยานพาหนะ**
2. สร้างกลุ่มสำหรับ:
   - การแชร์หลายสถานที่
   - การกระจายตามฤดูกาล
   - การจัดการล้น

#### การกำหนดค่าร้าน
อัปเดตรายละเอียดร้านของคุณ:
- ข้อมูลติดต่อ
- ที่อยู่และตำแหน่ง
- เวลาทำการ
- การตั้งค่าอีเมล

#### กฎราคา
จัดการการกำหนดราคาแบบไดนามิกสำหรับร้านของคุณภายใต้ **การตั้งค่า > กฎราคา**:

**กฎราคาคืออะไร?**
กฎราคาจะปรับอัตราค่าเช่าอัตโนมัติตาม:
- **ฤดูกาล** - ค่าธรรมเนียมเพิ่มเติมในไฮซีซัน (เช่น +30% พ.ย.-เม.ย.)
- **เหตุการณ์** - ราคาเทศกาล (เช่น +50% สงกรานต์)
- **วันในสัปดาห์** - พรีเมียมวันหยุดสุดสัปดาห์ (เช่น +10% วันเสาร์)

**เริ่มต้นอย่างรวดเร็วด้วยค่าที่ตั้งล่วงหน้าของประเทศไทย**
คลิก "สร้างค่าที่ตั้งล่วงหน้าของประเทศไทย" เพื่อเพิ่มกฎทั่วไป:
- ไฮซีซัน (15 พ.ย. - 15 เม.ย.): +30%
- เทศกาลสงกรานต์ (13-15 เม.ย.): +50%
- พรีเมียมวันหยุดสุดสัปดาห์ (วันเสาร์): +10%

**การสร้างกฎที่กำหนดเอง**
1. คลิก "+ เพิ่มกฎ"
2. ตั้งประเภทกฎ (ฤดูกาล เหตุการณ์ วันในสัปดาห์ กำหนดเอง)
3. กำหนดค่าช่วงวันที่หรือวัน
4. ตั้งตัวคูณ (เช่น 1.2 = +20%)
5. ตั้งอัตราต่ำสุด/สูงสุดเสริม
6. เลือกประเภทยานพาหนะที่จะใช้

**การจัดการกฎ**
- สลับสถานะใช้งาน/ไม่ใช้งาน
- แก้ไขกฎตามต้องการ
- ลบกฎที่ไม่เกี่ยวข้องอีกต่อไป
- กฎที่มีลำดับความสำคัญสูงกว่าจะมีผลก่อนเมื่อทับซ้อนกัน

## งานจัดการประจำวัน

### กิจวัตรตอนเช้า
- [ ] ตรวจสอบแดชบอร์ดสำหรับกิจกรรมข้ามคืน
- [ ] ตรวจสอบการคืนที่ครบกำหนดในวันนี้
- [ ] ตรวจสอบการเช่าที่เลยกำหนด
- [ ] แจ้งพนักงานเกี่ยวกับการจองของวัน

### ตลอดทั้งวัน
- [ ] ตรวจสอบการเช่าที่ใช้งานอยู่
- [ ] ตรวจสอบการเช็คเอาท์ที่เสร็จสิ้น
- [ ] จัดการปัญหาที่ส่งต่อมา
- [ ] ดำเนินการรายงานอุบัติเหตุ

### สิ้นวัน
- [ ] ตรวจสอบรายได้ประจำวัน
- [ ] ตรวจสอบสถานะเงินมัดจำ
- [ ] จัดการปัญหาที่ค้างอยู่
- [ ] วางแผนสำหรับการคืนวันพรุ่งนี้

## งานรายสัปดาห์

### ตรวจสอบรายได้
- สร้างรายงานการชำระเงิน
- วิเคราะห์แนวโน้มการเช่า
- เปรียบเทียบกับสัปดาห์ก่อน

### ตรวจสอบยานพาหนะ
- ตรวจสอบการใช้งานยานพาหนะ
- ตรวจสอบความต้องการการบำรุงรักษา
- วางแผนการหมุนเวียนยานพาหนะ

### ตรวจสอบทีม
- ตรวจสอบประสิทธิภาพของพนักงาน
- จัดการความต้องการการฝึกอบรม
- อัปเดตขั้นตอนตามต้องการ

## รายงาน

เข้าถึงรายงานผ่าน **การเงิน > รายงาน**:

| รายงาน | วัตถุประสงค์ |
|--------|---------|
| รายงานรายได้ | รายได้รายวัน/รายสัปดาห์/รายเดือน |
| รายงานการเช่า | สรุปกิจกรรมการเช่า |
| รายงานยานพาหนะ | การใช้งานยานพาหนะ |
| รายงานเงินมัดจำ | การติดตามสถานะเงินมัดจำ |

## แนวทางปฏิบัติที่ดี

### การบริการลูกค้า
1. ฝึกอบรมพนักงานเกี่ยวกับขั้นตอนที่ถูกต้อง
2. ตรวจสอบให้แน่ใจว่าราคาสม่ำเสมอ
3. จัดการข้อร้องเรียนทันที

### การจัดการยานพาหนะ
1. หมุนเวียนยานพาหนะอย่างสม่ำเสมอ
2. กำหนดเวลาการบำรุงรักษาเชิงป้องกัน
3. เก็บสต็อกอุปกรณ์เสริม

### การควบคุมการเงิน
1. กระทบยอดการชำระเงินทุกวัน
2. ตรวจสอบการจัดการเงินมัดจำ
3. ตรวจสอบราคาอย่างแข่งขัน

### การจัดการความเสี่ยง
1. ตรวจสอบให้แน่ใจว่าการเช่าทั้งหมดมีเอกสารที่ถูกต้อง
2. ตรวจสอบความคุ้มครองประกัน
3. บันทึกสภาพยานพาหนะ

## การแก้ไขปัญหา

### ปัญหาทั่วไป

**ยานพาหนะแสดงสถานะผิด**
- ตรวจสอบว่าการเช่าถูกปิดอย่างถูกต้องหรือไม่
- ยืนยันในส่วนการเช่า

**ความคลาดเคลื่อนของการชำระเงิน**
- ตรวจสอบบันทึกการชำระเงิน
- ตรวจสอบสถานะเงินมัดจำ

**ปัญหาการเข้าถึงของพนักงาน**
- ยืนยันการกำหนดบทบาท
- ติดต่อ OrgAdmin หากจำเป็น

## การขอความช่วยเหลือ

- **ปัญหาทางเทคนิค** - ติดต่อ OrgAdmin ของคุณ
- **คำถามเกี่ยวกับระบบ** - ตรวจสอบเอกสารคู่มือ
- **ปัญหาเกี่ยวกับบัญชี** - ติดต่อฝ่ายสนับสนุน
