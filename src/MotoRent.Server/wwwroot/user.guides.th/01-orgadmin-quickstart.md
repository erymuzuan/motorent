# คู่มือเริ่มต้นใช้งาน MotoRent - ผู้ดูแลองค์กร

ยินดีต้อนรับสู่ MotoRent! คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งานในฐานะผู้ดูแลองค์กร (OrgAdmin)

![แดชบอร์ด](images/02-dashboard.png)

## บทบาทของคุณ

ในฐานะ OrgAdmin คุณมี **สิทธิ์เข้าถึงเต็มรูปแบบ** สำหรับฟีเจอร์ทั้งหมดใน MotoRent คุณสามารถจัดการ:
- การเช่าและการเช็คอิน/เช็คเอาท์
- ยานพาหนะ (รถและอุปกรณ์เสริม)
- การเงิน (การชำระเงิน, เงินมัดจำ, รายงาน)
- อุบัติเหตุและเหตุการณ์
- ลูกค้า (ผู้เช่า)
- การตั้งค่า (ร้านค้า, ประกันภัย, กลุ่มยานพาหนะ)

## ภาพรวมการนำทาง

เมนูนำทางของคุณประกอบด้วย:

| เมนู | คำอธิบาย |
|------|-------------|
| **แดชบอร์ด** | ภาพรวมของการเช่าที่ใช้งานอยู่ ยานพาหนะที่พร้อมใช้งาน และการดำเนินการด่วน |
| **การเช่า** | จัดการการเช่าที่ใช้งานอยู่ เช็คอินลูกค้าใหม่ เช็คเอาท์เมื่อคืนรถ |
| **ตัวแทน** | ลงทะเบียนและติดตามพันธมิตรภายนอกและค่าคอมมิชชั่น |
| **การเงิน** | ติดตามการชำระเงิน เงินมัดจำ การจ่ายเงินเจ้าของ และสร้างรายงาน |
| **ยานพาหนะ** | จัดการยานพาหนะทุกประเภท (มอเตอร์ไซค์ รถยนต์ เรือ เจ็ตสกี รถตู้) |
| **อุบัติเหตุ** | รายงานและติดตามอุบัติเหตุและเหตุการณ์ของยานพาหนะ |
| **ลูกค้า** | จัดการข้อมูลและประวัติผู้เช่า |
| **การตั้งค่า** | กำหนดค่าร้านค้า ประกันภัย เทมเพลต และกฎราคา |

## เริ่มต้นใช้งาน

### 1. ภาพรวมแดชบอร์ด

เมื่อคุณล็อกอิน คุณจะเห็นแดชบอร์ดพร้อมด้วย:
- **การเช่าที่ใช้งานอยู่** - จำนวนยานพาหนะที่ให้เช่าอยู่ในปัจจุบัน
- **ยานพาหนะที่พร้อมใช้งาน** - ยานพาหนะที่พร้อมให้เช่า
- **ครบกำหนดวันนี้** - การคืนที่คาดว่าจะเกิดขึ้นวันนี้
- **รายได้วันนี้** - ยอดรวมที่เก็บได้วันนี้

**การดำเนินการด่วน** ทางด้านขวาให้คุณ:
- เริ่มเช็คอินใหม่
- ดำเนินการเช็คเอาท์
- ลงทะเบียนผู้เช่าใหม่
- จัดการยานพาหนะของคุณ

### 2. การดำเนินการเช่าใหม่ (เช็คอิน)

![วิซาร์ดเช็คอิน](images/04-checkin.png)

1. คลิก **การเช่า > เช็คอิน** หรือปุ่ม "เช็คอินใหม่"
2. ทำตามวิซาร์ด 5 ขั้นตอน:
   - **ขั้นตอน 1: ผู้เช่า** - ค้นหาผู้เช่าที่มีอยู่หรือสร้างใหม่ (รวม OCR สำหรับหนังสือเดินทาง/บัตรประชาชน)
   - **ขั้นตอน 2: ยานพาหนะ** - เลือกยานพาหนะที่ว่าง
   - **ขั้นตอน 3: กำหนดค่า** - ตั้งวันที่เช่า ประกันภัย อุปกรณ์เสริม
   - **ขั้นตอน 4: เงินมัดจำ** - บันทึกเงินมัดจำ (เงินสดหรือบัตร)
   - **ขั้นตอน 5: สัญญา** - ตรวจสอบและยืนยันสัญญาเช่า

### 3. การจัดการยานพาหนะของคุณ

![การจัดการยานพาหนะ](images/05-fleet.png)

ไปที่ **ยานพาหนะ > ยานพาหนะทั้งหมด** เพื่อ:
- ดูยานพาหนะทั้งหมดทุกประเภท (มอเตอร์ไซค์ รถยนต์ เจ็ตสกี เรือ รถตู้)
- กรองตามสถานะ (พร้อมใช้งาน เช่าอยู่ ซ่อมบำรุง)
- เพิ่มยานพาหนะใหม่ด้วยปุ่ม "+ เพิ่มยานพาหนะ"
- แก้ไขรายละเอียดยานพาหนะ ราคา และสถานะ

**สีสถานะยานพาหนะ:**
- **เขียว (พร้อมใช้งาน)** - พร้อมให้เช่า
- **น้ำเงิน (เช่าอยู่)** - กำลังถูกใช้งานโดยลูกค้า
- **ส้ม (ซ่อมบำรุง)** - กำลังได้รับการบริการ

### 4. การจัดการตัวแทน

เชื่อมต่อกับโรงแรมและบริษัทท่องเที่ยวเพื่อขยายธุรกิจของคุณ:
- **ลงทะเบียนตัวแทน**: ติดตามข้อมูลติดต่อและอัตราค่าคอมมิชชั่น
- **เชื่อมโยงการจอง**: คำนวณค่าคอมมิชชั่นอัตโนมัติเมื่อตัวแทนแนะนำลูกค้า
- **ติดตามการจ่ายเงิน**: บันทึกและจัดการการจ่ายเงินให้ตัวแทนผ่านลิ้นชักเงินสด

ดู [คู่มือการจัดการตัวแทน](10-agent-management-guide.md) สำหรับรายละเอียดเพิ่มเติม

### 5. การจัดการการเงิน

#### การชำระเงิน
- ติดตามการชำระเงินค่าเช่าทั้งหมด (เงินสด บัตร PromptPay โอนเงินผ่านธนาคาร)
- กรองตามสถานะ วิธีการ และช่วงวันที่
- บันทึกการชำระเงินด้วยตนเอง

#### เงินมัดจำ
- ดูเงินมัดจำที่ถืออยู่
- ดำเนินการคืนเงินเมื่อการเช่าเสร็จสิ้น
- ติดตามเงินมัดจำที่ถูกริบ

#### การจ่ายเงินเจ้าของ
- หากคุณจัดการยานพาหนะสำหรับเจ้าของภายนอก ติดตามรายได้และดำเนินการจ่ายเงินรายเดือน

### 6. การจัดการอุบัติเหตุ

![การจัดการอุบัติเหตุ](images/06-accidents.png)

เมื่อเกิดอุบัติเหตุ:
1. ไปที่ **อุบัติเหตุ**
2. คลิก "รายงานอุบัติเหตุ"
3. กรอกรายละเอียดเหตุการณ์ในแท็บใหม่:
   - **รายละเอียด**: วันที่ เวลา สถานที่ และคำอธิบาย
   - **ฝ่ายที่เกี่ยวข้อง**: ลงทะเบียนยานพาหนะหรือบุคคลอื่นที่เกี่ยวข้อง
   - **ค่าใช้จ่าย**: ติดตามการประเมินค่าซ่อมและค่าใช้จ่ายจริง
   - **เอกสาร**: อัปโหลดรายงานตำรวจ เคลมประกัน และรูปถ่าย
4. ติดตามความคืบหน้าการแก้ไขและการจ่ายเงินประกัน

### 7. การปรับแต่งเอกสารของคุณ

ใช้ **เครื่องมือออกแบบเทมเพลตเอกสาร** เพื่อทำให้ MotoRent ดูเหมือนแบรนด์ของคุณเอง:
- **โปรแกรมแก้ไขแบบภาพ**: ลากและวางบล็อกเพื่อออกแบบส่วนหัวและส่วนท้ายของคุณเอง
- **ข้อมูลไดนามิก**: ใช้โทเค็นเช่น `{{CustomerName}}` เพื่อกรอกข้อมูลการเช่าอัตโนมัติ
- **เทมเพลตมาตรฐาน**: สร้างเวอร์ชันที่กำหนดเองของการยืนยันการจอง สัญญาเช่า และใบเสร็จ

ดู [คู่มือเครื่องมือออกแบบเทมเพลตเอกสาร](09-template-designer-guide.md) สำหรับรายละเอียดเพิ่มเติม

### 8. ร้านค้าและสถานที่ให้บริการ

กำหนดค่าเครือข่ายการเช่าของคุณภายใต้ **การตั้งค่า**:

#### การตั้งค่าร้านค้า
- เพิ่มสถานที่ร้านค้าจริงใหม่
- ตั้งเวลาทำการและข้อมูลติดต่อ

#### สถานที่ให้บริการ
- จัดการจุดรับและส่ง (โรงแรม สนามบิน)
- ตั้ง **ค่าธรรมเนียมการส่ง** สำหรับการเช่าแบบทางเดียว

ดู [คู่มือสถานที่ให้บริการ](11-service-locations-guide.md) สำหรับรายละเอียดเพิ่มเติม

### 9. การกำหนดราคาแบบไดนามิกและปฏิทิน

ปรับอัตราค่าเช่าอัตโนมัติตามฤดูกาล เหตุการณ์ และวันในสัปดาห์

ไปที่ **การตั้งค่า > กฎราคา** เพื่อจัดการการกำหนดราคาแบบไดนามิก:

#### ปฏิทินราคา
ใช้ **ปฏิทินราคา** แบบภาพเพื่อดูว่ากฎใดใช้งานอยู่สำหรับวันที่และประเภทยานพาหนะที่กำหนด

#### การสร้างกฎราคา
1. คลิก "+ เพิ่มกฎ"
2. กำหนดค่ากฎ:
   - **ชื่อกฎ** - ชื่อที่อธิบายได้ (เช่น "ไฮซีซัน" "เทศกาลสงกรานต์")
   - **ประเภทกฎ** - ฤดูกาล เหตุการณ์ วันในสัปดาห์ หรือกำหนดเอง
   - **วันที่** - วันเริ่มต้นและสิ้นสุด (หรือเลือกวันสำหรับกฎรายสัปดาห์)
   - **ตัวคูณ** - การปรับราคา (เช่น 1.3 = +30%, 0.9 = -10%)
   - **อัตราต่ำสุด/สูงสุด** - ขีดจำกัดอัตราเสริม
   - **ลำดับความสำคัญ** - กฎที่มีลำดับความสำคัญสูงกว่าจะใช้ก่อนเมื่อทับซ้อนกัน
   - **ประเภทยานพาหนะ** - ใช้กับประเภทเฉพาะหรือยานพาหนะทั้งหมด

#### ค่าที่ตั้งล่วงหน้าของประเทศไทย
คลิก "สร้างค่าที่ตั้งล่วงหน้าของประเทศไทย" เพื่อสร้างกฎทั่วไปทันที:
- **ไฮซีซัน** (15 พ.ย. - 15 เม.ย.): +30%
- **เทศกาลสงกรานต์** (13-15 เม.ย.): +50%
- **พรีเมียมวันหยุดสุดสัปดาห์** (วันเสาร์): +10%

#### ประเภทกฎ
| ประเภท | คำอธิบาย |
|------|-------------|
| **ฤดูกาล** | ช่วงวันที่ที่เกิดซ้ำ (เช่น ไฮซีซัน) |
| **เหตุการณ์** | เหตุการณ์หรือเทศกาลครั้งเดียว |
| **วันในสัปดาห์** | รูปแบบรายสัปดาห์ (เช่น ราคาวันหยุดสุดสัปดาห์) |
| **กำหนดเอง** | ช่วงวันที่เฉพาะ |

#### ตัวอย่างราคา
กล่องโต้ตอบจะแสดงตัวอย่างแบบสดว่าตัวคูณส่งผลต่อราคาอย่างไร:
- อัตราพื้นฐาน: 500/วัน พร้อมตัวคูณ 1.3 เท่า = 650/วัน (+30%)

## เคล็ดลับสู่ความสำเร็จ

1. **งานประจำวัน:**
   - ตรวจสอบแดชบอร์ดสำหรับการคืนที่ครบกำหนด
   - ดำเนินการเช็คเอาท์ทันที
   - ตรวจสอบการเช่าที่ค้างชำระ

2. **งานประจำสัปดาห์:**
   - ตรวจสอบรายงานอุบัติเหตุ
   - ตรวจสอบสถานะการบำรุงรักษายานพาหนะ
   - สร้างรายงานรายได้

3. **แนวทางปฏิบัติที่ดี:**
   - ถ่ายรูปก่อนและหลังการเช่าเสมอ
   - ตรวจสอบเอกสารผู้เช่า (หนังสือเดินทาง/บัตรประชาชน)
   - อัปเดตแพ็กเกจประกันให้เป็นปัจจุบัน

## การขอความช่วยเหลือ

- คลิก **เอกสารคู่มือ** ที่ส่วนท้ายสำหรับคู่มือโดยละเอียด
- ติดต่อฝ่ายสนับสนุนผ่านลิงก์ช่วยเหลือ
- ตรวจสอบกับผู้ดูแลระบบของคุณสำหรับปัญหาเกี่ยวกับบัญชี
