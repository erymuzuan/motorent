# คู่มือพอร์ทัลนักท่องเที่ยว MotoRent

ยินดีต้อนรับสู่ MotoRent! คู่มือนี้จะช่วยคุณเรียกดูและเช่ายานพาหนะระหว่างวันหยุดพักผ่อนในประเทศไทย

## เริ่มต้นใช้งาน

### การเข้าถึงพอร์ทัล

1. เยี่ยมชมเว็บไซต์ร้านเช่าของคุณ (จัดหาโดยโรงแรมหรือตัวแทนท่องเที่ยว)
2. เรียกดูยานพาหนะที่พร้อมให้เช่าโดยไม่ต้องลงชื่อเข้าใช้
3. สร้างบัญชีหรือลงชื่อเข้าใช้เพื่อทำการจอง

### การเรียกดูยานพาหนะ

หน้าหลักแสดงยานพาหนะที่พร้อมให้เช่าพร้อมด้วย:
- **ประเภทยานพาหนะ** - มอเตอร์ไซค์ รถยนต์ เจ็ตสกี เรือ รถตู้
- **อัตราต่อวัน** - ราคาต่อวันในบาท (THB)
- **ความพร้อมใช้งาน** - สถานะแบบเรียลไทม์

ใช้ตัวกรองเพื่อจำกัดการค้นหาของคุณ:
- ประเภทยานพาหนะ
- ช่วงราคา
- คุณสมบัติ (ออโตเมติก/เกียร์ธรรมดา)

## การจองล่วงหน้า

### ขั้นตอน 1: เลือกยานพาหนะ

1. คลิกที่การ์ดยานพาหนะเพื่อดูรายละเอียด
2. ตรวจสอบ:
   - รูปถ่ายและข้อมูลจำเพาะ
   - อัตราต่อวันและจำนวนเงินมัดจำ
   - ตัวเลือกประกัน
   - อุปกรณ์เสริมที่รวมอยู่ (หมวกกันน็อค ฯลฯ)

### ขั้นตอน 2: เลือกวันที่

1. เลือกวันที่และเวลารับรถ
2. เลือกวันที่และเวลาคืนรถ
3. ตรวจสอบระยะเวลาเช่าทั้งหมด
4. ดูค่าใช้จ่ายทั้งหมดโดยประมาณ

### ขั้นตอน 3: เพิ่มตัวเลือก

1. **ประกัน** - เลือกจากแพ็กเกจที่มี:
   - พื้นฐาน (รวมอยู่แล้ว)
   - มาตรฐาน - ค่าเสียหายส่วนแรกต่ำกว่า
   - พรีเมียม - ความคุ้มครองเต็มรูปแบบ
2. **อุปกรณ์เสริม** - เพิ่มรายการเสริม:
   - หมวกกันน็อคเพิ่มเติม
   - ที่ยึดโทรศัพท์
   - อุปกรณ์ GPS

### ขั้นตอน 4: ข้อมูลของคุณ

ให้รายละเอียดของคุณ:
- ชื่อเต็ม (ตามหนังสือเดินทาง)
- สัญชาติ
- หมายเลขหนังสือเดินทาง
- หมายเลขโทรศัพท์ (สำหรับ WhatsApp/SMS)
- ที่อยู่อีเมล
- ชื่อโรงแรม/ที่พัก

### ขั้นตอน 5: ยืนยันการจอง

1. ตรวจสอบสรุปการจองของคุณ
2. ยอมรับข้อกำหนดและเงื่อนไขการเช่า
3. ส่งการจองของคุณ
4. รับการยืนยันทางอีเมล

## ที่ร้านเช่า

### ขั้นตอนเช็คอิน

เมื่อคุณมาถึงร้าน:

1. **นำเสนอเอกสาร**
   - หนังสือเดินทางหรือบัตรประชาชน
   - ใบขับขี่ (แนะนำใบขับขี่สากล)
   - การยืนยันการจอง

2. **จ่ายเงินมัดจำ**
   - เงินสด (THB) หรือบัตรเครดิต
   - เงินมัดจำจะได้รับคืนเมื่อคืนรถ

3. **การตรวจสอบยานพาหนะ**
   - พนักงานจะพาคุณชมยานพาหนะ
   - ถ่ายรูปสภาพปัจจุบัน
   - ทดสอบยานพาหนะก่อนออกเดินทาง

4. **รับกุญแจ**
   - ลงนามในสัญญาเช่า
   - รับหมายเลขติดต่อฉุกเฉิน
   - รับยานพาหนะ

### เอกสารสำคัญ

**จำเป็น:**
- หนังสือเดินทางที่ยังไม่หมดอายุ
- ใบขับขี่
- ใบขับขี่สากล (แนะนำสำหรับมอเตอร์ไซค์)

**จัดหาให้:**
- สัญญาเช่า
- ใบรับรองประกัน
- บัตรติดต่อฉุกเฉิน

## ระหว่างการเช่า

### เคล็ดลับความปลอดภัย

1. **สวมหมวกกันน็อคเสมอ** (ตามกฎหมายกำหนด)
2. **ขับขี่ทางซ้าย** ของถนน
3. **เคารพขีดจำกัดความเร็ว**
4. **หลีกเลี่ยงการขับขี่ตอนกลางคืน** หากไม่คุ้นเคยกับถนน
5. **พกเอกสารติดตัว** ตลอดเวลา

### ในกรณีที่เกิดปัญหา

**เสียหรือปัญหาเครื่องยนต์:**
1. เคลื่อนไปยังสถานที่ปลอดภัย
2. โทรหาหมายเลขฉุกเฉินบนบัตรติดต่อของคุณ
3. รอความช่วยเหลือ

**อุบัติเหตุ:**
1. ตรวจสอบให้แน่ใจว่าทุกคนปลอดภัย
2. โทรเรียกบริการฉุกเฉินหากจำเป็น (191)
3. ติดต่อร้านเช่าทันที
4. ถ่ายรูปที่เกิดเหตุ
5. ขอรายงานตำรวจหากจำเป็น

**กุญแจหาย:**
1. ติดต่อร้านเช่า
2. อาจมีค่าธรรมเนียมเปลี่ยนทดแทน

## การคืนยานพาหนะ

### ขั้นตอนเช็คเอาท์

1. **คืนตรงเวลา**
   - การคืนล่าช้าอาจมีค่าใช้จ่ายเพิ่มเติม
   - ติดต่อร้านหากต้องการขยายเวลา

2. **การตรวจสอบยานพาหนะ**
   - พนักงานจะตรวจสอบยานพาหนะ
   - เปรียบเทียบสภาพกับรูปถ่ายเช็คอิน
   - บันทึกความเสียหายใหม่ใดๆ

3. **การชำระเงินสุดท้าย**
   - วันเพิ่มเติม (ถ้ามี)
   - ค่าน้ำมัน (ถ้าไม่เต็ม)
   - ค่าความเสียหาย (ถ้ามี)

4. **การคืนเงินมัดจำ**
   - เงินมัดจำเงินสดคืนทันที
   - เงินมัดจำบัตรคืนภายใน 3-7 วัน

ลงชื่อเข้าใช้บัญชีของคุณเพื่อดู:
- การจองทั้งหมดในอดีต
- รายละเอียดการเช่าและใบเสร็จ
- ประวัติการชำระเงิน

### การแสดงความคิดเห็น

หลังจากการเช่าของคุณ:
1. ให้คะแนนประสบการณ์ของคุณ
2. แสดงความคิดเห็น
3. ช่วยเหลือนักเดินทางคนอื่น

## คำถามที่พบบ่อย

### ฉันต้องมีใบขับขี่หรือไม่?

**มอเตอร์ไซค์:** ใบขับขี่สากลหรือใบขับขี่ที่ถูกต้องจากประเทศของคุณ
**รถยนต์:** ใบขับขี่ที่ถูกต้องจากประเทศของคุณ
**เจ็ตสกี/เรือ:** ไม่ต้องมีใบอนุญาต แต่มีการฝึกอบรมสั้นๆ ให้

### ถ้าฉันทำยานพาหนะเสียหายล่ะ?

ค่าความเสียหายขึ้นอยู่กับ:
- ความคุ้มครองประกันที่เลือก
- ความรุนแรงของความเสียหาย
- จำนวนค่าเสียหายส่วนแรก

รายงานความเสียหายทั้งหมดทันทีเพื่อลดค่าใช้จ่าย

### ฉันสามารถขยายการเช่าได้หรือไม่?

ได้! ติดต่อร้าน:
- ก่อนวันที่คืน
- การขยายเวลาขึ้นอยู่กับความพร้อมใช้งาน
- ต้องชำระเงินเพิ่มเติม

### อะไรรวมอยู่ในการเช่า?

การเช่ามาตรฐานรวม:
- หมวกกันน็อคหนึ่งใบ (มอเตอร์ไซค์)
- ประกันพื้นฐาน
- บริการฉุกเฉินตลอด 24 ชั่วโมง
- ระยะทางไม่จำกัด (ในพื้นที่)

### ฉันจ่ายเงินได้อย่างไร?

วิธีการชำระเงินที่รับได้:
- เงินสด (THB)
- บัตรเครดิต/เดบิต (Visa, Mastercard)
- PromptPay (การชำระเงินผ่านมือถือของไทย)
