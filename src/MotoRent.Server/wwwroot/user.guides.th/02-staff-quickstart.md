# คู่มือเริ่มต้นใช้งาน MotoRent - พนักงาน

ยินดีต้อนรับสู่ MotoRent! คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งานในฐานะพนักงาน

![แดชบอร์ดพนักงาน](images/09-staff-dashboard.png)

## บทบาทของคุณ

ในฐานะพนักงาน คุณจัดการการดำเนินงานเช่าประจำวัน:
- ดำเนินการเช็คอิน (การเช่าใหม่)
- ดำเนินการเช็คเอาท์ (การคืน)
- จัดการการเช่าที่ใช้งานอยู่
- ลงทะเบียนลูกค้าใหม่
- รายงานอุบัติเหตุ

## ภาพรวมการนำทาง

เมนูนำทางของคุณประกอบด้วย:

| เมนู | คำอธิบาย |
|------|-------------|
| **แดชบอร์ด** | ภาพรวมของการเช่าที่ใช้งานอยู่และการดำเนินการด่วน |
| **การเช่า** | จัดการการเช่าที่ใช้งานอยู่ เช็คอิน เช็คเอาท์ |
| **อุบัติเหตุ** | รายงานและดูอุบัติเหตุของยานพาหนะ |
| **ลูกค้า** | ดูและลงทะเบียนผู้เช่า |

> **หมายเหตุ:** การเงิน การจัดการยานพาหนะ และการตั้งค่าจัดการโดยผู้จัดการร้านและผู้ดูแล

## เริ่มต้นใช้งาน

### 1. ภาพรวมแดชบอร์ด

แดชบอร์ดของคุณแสดง:
- **การเช่าที่ใช้งานอยู่** - ยานพาหนะที่ให้เช่าอยู่ในปัจจุบัน
- **ยานพาหนะที่พร้อมใช้งาน** - พร้อมให้เช่า
- **ครบกำหนดวันนี้** - การคืนที่คาดหวัง
- **รายได้วันนี้** - การเก็บเงิน (ดูอย่างเดียว)

**การดำเนินการด่วน:**
- **เช็คอินใหม่** - เริ่มการเช่าใหม่
- **ดำเนินการเช็คเอาท์** - คืนยานพาหนะ
- **ลงทะเบียนผู้เช่า** - เพิ่มลูกค้าใหม่

### 2. การดำเนินการเช่าใหม่ (เช็คอิน)

![กระบวนการเช็คอิน](images/04-checkin.png)

นี่คืองานที่พบบ่อยที่สุดของคุณ ทำตามขั้นตอนเหล่านี้:

#### ขั้นตอน 1: เลือกผู้เช่า
1. คลิก **การเช่า > เช็คอิน** หรือ "เช็คอินใหม่"
2. ค้นหาลูกค้าด้วย:
   - ชื่อ
   - หมายเลขโทรศัพท์
   - หมายเลขหนังสือเดินทาง
3. หากเป็นลูกค้าใหม่ คลิก "สร้างผู้เช่าใหม่"

#### ขั้นตอน 2: เลือกยานพาหนะ
1. เรียกดูยานพาหนะที่พร้อมใช้งาน
2. ใช้ตัวกรองเพื่อจำกัดขอบเขต:
   - ประเภทยานพาหนะ (มอเตอร์ไซค์ รถยนต์ ฯลฯ)
   - ยี่ห้อ
3. คลิกที่ยานพาหนะเพื่อเลือก

#### ขั้นตอน 3: กำหนดค่าการเช่า
1. ตั้งวันที่เช่า:
   - วันเริ่มต้น (ปกติคือวันนี้)
   - วันสิ้นสุดที่คาดหวัง
2. เพิ่มแพ็กเกจประกันเสริม
3. เพิ่มอุปกรณ์เสริม (หมวกกันน็อค ที่ยึดโทรศัพท์ ฯลฯ)

#### ขั้นตอน 4: เก็บเงินมัดจำ
1. ป้อนจำนวนเงินมัดจำ
2. เลือกประเภทเงินมัดจำ:
   - เงินสด
   - บัตร
3. บันทึกเงินมัดจำ

#### ขั้นตอน 5: ยืนยันสัญญา
1. ตรวจสอบสรุปการเช่า
2. พิมพ์หรือส่งอีเมลสัญญา
3. คลิก "เช็คอินเสร็จสิ้น"

### 3. การดำเนินการคืน (เช็คเอาท์)

เมื่อลูกค้าคืนยานพาหนะ:

1. ไปที่ **การเช่า** และค้นหาการเช่าที่ใช้งานอยู่
2. คลิก **เช็คเอาท์** บนการ์ดการเช่า
3. ดำเนินการตรวจสอบ:
   - ตรวจสอบสภาพยานพาหนะ
   - บันทึกระดับน้ำมัน
   - บันทึกความเสียหายใหม่ใดๆ
4. คำนวณค่าใช้จ่ายสุดท้าย:
   - วันเพิ่มเติม (ถ้ามี)
   - ค่าความเสียหาย
   - ค่าปรับล่าช้า
5. ดำเนินการคืนเงินหรือเก็บเงินเพิ่มเติม
6. ปล่อยเงินมัดจำ

### 4. การจัดการการเช่าที่ใช้งานอยู่

หน้า **การเช่า** แสดงการเช่าทั้งหมดพร้อมตัวกรอง:

| ตัวกรอง | คำอธิบาย |
|--------|-------------|
| **ใช้งานอยู่** | กำลังให้เช่าอยู่ในปัจจุบัน |
| **ครบกำหนดวันนี้** | คาดว่าจะคืนวันนี้ |
| **เลยกำหนด** | เลยวันที่คาดว่าจะคืน |
| **เสร็จสิ้น** | คืนเรียบร้อยแล้ว |

**การดำเนินการบนการ์ดการเช่า:**
- **เช็คเอาท์** - ดำเนินการคืน
- **ขยายเวลา** - เพิ่มวันเพิ่มเติม
- **ยกเลิก** - ยกเลิกการเช่า
- **ใบแจ้งหนี้** - ดู/พิมพ์ใบแจ้งหนี้

### 5. การรายงานอุบัติเหตุ

หากลูกค้ารายงานความเสียหายหรืออุบัติเหตุ:
1. ไปที่ **อุบัติเหตุ**
2. คลิก "รายงานอุบัติเหตุ"
3. กรอกรายละเอียดเหตุการณ์:
   - **รายละเอียด**: ยานพาหนะ การเช่า วันที่ และคำอธิบาย
   - **ฝ่ายที่เกี่ยวข้อง**: เพิ่มบุคคลหรือยานพาหนะอื่นที่เกี่ยวข้อง
   - **รูปถ่าย**: อัปโหลดรูปถ่ายความเสียหาย
4. หากมีค่าซ่อม ป้อนในแท็บ **ค่าใช้จ่าย**
5. บันทึกรายงาน สถานะยานพาหนะจะเปลี่ยนเป็น **ซ่อมบำรุง** โดยอัตโนมัติหากความเสียหายรุนแรง

### 6. การลงทะเบียนลูกค้าใหม่

เพื่อเพิ่มผู้เช่าใหม่:

1. ไปที่ **ลูกค้า > ผู้เช่า**
2. คลิก "ลงทะเบียนผู้เช่า"
3. กรอกข้อมูลที่จำเป็น:
   - ชื่อเต็ม
   - สัญชาติ
   - หมายเลขหนังสือเดินทางหรือบัตรประชาชน
   - หมายเลขโทรศัพท์
   - อีเมล
   - โรงแรม/ที่พัก
4. บันทึกข้อมูล

## ขั้นตอนการทำงานประจำวัน

### รายการตรวจสอบตอนเช้า
- [ ] ตรวจสอบแดชบอร์ดสำหรับการคืนที่ครบกำหนด
- [ ] ตรวจสอบการเช่าที่เลยกำหนด
- [ ] เตรียมยานพาหนะที่พร้อมใช้งาน

### ระหว่างวัน
- [ ] ดำเนินการเช็คอินทันที
- [ ] ตรวจสอบเอกสารลูกค้า
- [ ] ถ่ายรูปยานพาหนะก่อนเช่า
- [ ] บันทึกสภาพยานพาหนะเมื่อคืน

### สิ้นวัน
- [ ] ตรวจสอบการเช่าที่ยังไม่เสร็จสิ้น
- [ ] รายงานเหตุการณ์ใดๆ ให้ผู้จัดการทราบ
- [ ] บันทึกยานพาหนะที่ต้องการความสนใจ

## เคล็ดลับสู่ความสำเร็จ

1. **ตรวจสอบเอกสารเสมอ** - ตรวจสอบความถูกต้องของหนังสือเดินทาง/บัตรประชาชน
2. **ถ่ายรูป** - บันทึกสภาพยานพาหนะก่อนและหลัง
3. **อธิบายเงื่อนไขให้ชัดเจน** - อธิบายวันที่เช่า เงินมัดจำ ประกันภัย
4. **จัดระเบียบ** - ใช้ตัวกรองเพื่อค้นหาการเช่าอย่างรวดเร็ว
5. **รายงานปัญหาทันที** - อย่าชะลอการรายงานอุบัติเหตุ

## ต้องการความช่วยเหลือ?

- ติดต่อผู้จัดการร้านของคุณสำหรับ:
  - ปัญหาการชำระเงิน
  - คำถามเกี่ยวกับราคา
  - ข้อกังวลเรื่องความพร้อมของยานพาหนะ

- ติดต่อผู้ดูแลสำหรับ:
  - ปัญหาการเข้าถึงระบบ
  - ปัญหาเกี่ยวกับบัญชี
