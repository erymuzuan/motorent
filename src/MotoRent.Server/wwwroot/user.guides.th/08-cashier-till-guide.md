# คู่มือระบบแคชเชียร์

คู่มือนี้ครอบคลุมระบบลิ้นชักเงินสดของแคชเชียร์สำหรับจัดการการดำเนินงานลิ้นชักเงินสด การชำระเงิน การจ่ายเงิน และการกระทบยอดกะการทำงาน

## ภาพรวม

ระบบลิ้นชักเงินสดเป็นศูนย์กลางสำหรับธุรกรรมทางการเงินทั้งหมดระหว่างกะการทำงานของคุณ ก่อนดำเนินการเช่าใดๆ คุณต้องเปิดเซสชันลิ้นชักเงินสด การชำระเงิน การคืนเงิน และการจ่ายเงินทั้งหมดจะถูกติดตามผ่านลิ้นชักเงินสดของคุณ

**แนวคิดหลัก:**
- **เซสชันลิ้นชักเงินสด** - เซสชันลิ้นชักเงินสดส่วนตัวของคุณสำหรับวัน
- **เงินสดเปิด** - จำนวนเงินสดเริ่มต้นในลิ้นชักของคุณ
- **เงินเข้า** - เงินที่ได้รับ (การชำระเงินค่าเช่า เงินมัดจำ)
- **เงินออก** - เงินที่จ่ายออก (การคืนเงิน น้ำมัน ค่าคอมมิชชั่น)
- **การกระทบยอด** - การสมดุลลิ้นชักของคุณเมื่อสิ้นสุดกะ

## เริ่มต้นใช้งาน

### การเปิดลิ้นชักเงินสดของคุณ

คุณต้องเปิดลิ้นชักเงินสดก่อนดำเนินการเช็คอินหรือเช็คเอาท์ใดๆ

1. คลิกปุ่ม **เปิดลิ้นชักเงินสด** ในส่วนหัว (หรือไปที่ `/staff/till`)
2. เลือก **ร้าน** ที่คุณทำงานอยู่ (ถ้าคุณมีสิทธิ์เข้าถึงหลายร้าน)
3. ป้อนจำนวน **เงินสดเปิด** (เงินสดที่คุณเริ่มต้นด้วย)
4. เพิ่ม **หมายเหตุ** (ไม่บังคับ)
5. คลิก **เปิดเซสชัน**

**กฎสำคัญ:**
- คุณมีได้เพียงหนึ่งลิ้นชักเงินสดที่เปิดต่อร้านต่อวัน
- คุณต้องปิดเซสชันปัจจุบันก่อนเปิดใหม่
- ร้านจะถูกล็อคกับลิ้นชักเงินสดของคุณ - ธุรกรรมทั้งหมดผ่านมัน

### แดชบอร์ดลิ้นชักเงินสด

เมื่อเปิดลิ้นชักเงินสดแล้ว แดชบอร์ดจะแสดง:

| ส่วน | คำอธิบาย |
|---------|-------------|
| **ข้อมูลเซสชัน** | ชื่อของคุณ ร้าน เวลาเริ่มต้น |
| **เงินเข้า** | เงินทั้งหมดที่ได้รับในเซสชันนี้ |
| **เงินออก** | เงินทั้งหมดที่จ่ายออกในเซสชันนี้ |
| **ยอดคงเหลือที่คาดหวัง** | เงินสดเปิด + เงินเข้า - เงินออก |
| **การดำเนินการด่วน** | จ่ายเงิน ฝาก เติมเงิน ปิด |
| **ธุรกรรมล่าสุด** | 10 ธุรกรรมล่าสุด |

## การบันทึกธุรกรรม

### ธุรกรรมอัตโนมัติ

เหล่านี้ถูกบันทึกอัตโนมัติเมื่อคุณดำเนินการเช่า:

| ประเภทธุรกรรม | เมื่อเกิดขึ้น |
|-----------------|-----------------|
| **การชำระเงินค่าเช่า** | ลูกค้าชำระเงินสำหรับการเช่าที่เช็คอิน |
| **เงินมัดจำ** | ลูกค้าชำระเงินมัดจำเงินสด |
| **การชำระเงินบัตร** | ลูกค้าชำระเงินด้วยบัตร (ติดตาม ไม่กระทบเงินสด) |
| **PromptPay** | ลูกค้าชำระเงินผ่าน PromptPay (ติดตาม ไม่กระทบเงินสด) |
| **การคืนเงินมัดจำ** | การคืนเงินสดที่เช็คเอาท์ |
| **ค่าความเสียหาย** | ลูกค้าชำระเงินสำหรับความเสียหาย |

### การจ่ายเงินด้วยตนเอง

ใช้ปุ่ม **บันทึกการจ่ายเงิน** สำหรับเงินสดที่ออกจากลิ้นชักของคุณ

#### การคืนเงินค่าน้ำมัน
เมื่อคุณคืนเงินลูกค้าสำหรับค่าน้ำมัน:
1. คลิก **บันทึกการจ่ายเงิน**
2. เลือก **คืนเงินค่าน้ำมัน**
3. ป้อนจำนวนเงิน
4. ป้อนชื่อลูกค้า/ผู้รับ
5. ป้อนหมายเลขใบเสร็จ (ถ้ามี)
6. เพิ่มหมายเหตุ (เช่น "เติมน้ำมันเต็มถังสำหรับการเช่า #123")
7. คลิก **บันทึก**

#### ค่าคอมมิชชั่นตัวแทน
เมื่อจ่ายค่าคอมมิชชั่นให้ตัวแทนที่นำลูกค้ามา:
1. คลิก **บันทึกการจ่ายเงิน**
2. เลือก **ค่าคอมมิชชั่นตัวแทน**
3. ป้อนจำนวนเงิน
4. ป้อนชื่อตัวแทน
5. อ้างอิงการจองหรือการเช่า
6. คลิก **บันทึก**

#### เงินสดย่อย
สำหรับค่าใช้จ่ายเบ็ดเตล็ด:
1. คลิก **บันทึกการจ่ายเงิน**
2. เลือก **เงินสดย่อย**
3. ป้อนจำนวนเงิน
4. อธิบายว่าใช้สำหรับอะไร (เช่น "อุปกรณ์สำนักงาน")
5. ป้อนหมายเลขใบเสร็จ
6. คลิก **บันทึก**

**เก็บใบเสร็จสำหรับการจ่ายเงินทุกครั้งเสมอ!**

## การรับการชำระเงินจากลิ้นชักเงินสด

หน้าลิ้นชักเงินสดมีส่วน **การชำระเงินด่วน** สำหรับรับการชำระเงินโดยตรงโดยไม่ต้องผ่านขั้นตอนเช็คอินเต็มรูปแบบ

### ปุ่มการชำระเงินด่วน

| ปุ่ม | ใช้สำหรับ |
|--------|---------|
| **การชำระเงินค่าเช่า** | การชำระเงินเพิ่มเติมสำหรับการเช่าที่มีอยู่ |
| **เงินมัดจำ** | เงินมัดจำค้ำประกันสำหรับการเช่าที่ใช้งานอยู่ |
| **เงินมัดจำการจอง** | เงินมัดจำสำหรับการจองล่วงหน้า |

### การบันทึกการชำระเงินค่าเช่า

สำหรับเมื่อลูกค้าต้องการชำระเงินเพิ่มเติมสำหรับการเช่าที่มีอยู่:

1. ที่หน้าลิ้นชักเงินสดของคุณ คลิก **การชำระเงินค่าเช่า**
2. ค้นหาการเช่าด้วย ID หรือชื่อลูกค้า
3. เลือกการเช่าจากผลลัพธ์
4. ป้อนจำนวนเงินที่ชำระ
5. เลือกวิธีการชำระเงิน (เงินสด บัตร PromptPay โอนเงินผ่านธนาคาร)
6. คลิก **บันทึกการชำระเงิน**
7. การชำระเงินปรากฏในธุรกรรมของคุณ

### การบันทึกเงินมัดจำการจอง

เมื่อลูกค้าชำระเงินมัดจำสำหรับการจองล่วงหน้า:

1. คลิก **เงินมัดจำการจอง**
2. ป้อนรหัสอ้างอิงการจอง (รหัส 6 ตัวอักษรเช่น "ABC123")
3. หรือค้นหาด้วยชื่อลูกค้า
4. ตรวจสอบรายละเอียดการจอง (วันที่ เงินมัดจำที่ต้องการ)
5. ป้อนจำนวนเงินมัดจำ (ไม่เกินยอดค้างชำระ)
6. เลือกวิธีการชำระเงิน
7. คลิก **บันทึกการชำระเงิน**
8. ใบเสร็จถูกสร้างอัตโนมัติสำหรับลูกค้า

### การฝากเงินสด

เมื่อคุณมีเงินสดในลิ้นชักมากเกินไปและต้องย้ายไปยังตู้เซฟ:

1. คลิก **ฝากเงินสด**
2. ป้อนจำนวนเงินที่ฝาก
3. เพิ่มหมายเหตุ (เช่น "ฝากบ่ายไปตู้เซฟ")
4. คลิก **ยืนยัน**

### เติมเงิน

หากคุณต้องการเงินสดเพิ่มในลิ้นชัก (เช่น สำหรับทอนเงิน):

1. คลิก **เติมเงิน**
2. ป้อนจำนวนเงินที่เพิ่ม
3. เพิ่มหมายเหตุ (เช่น "เงินทอนจากตู้เซฟ")
4. คลิก **ยืนยัน**

## การปิดลิ้นชักเงินสดของคุณ

เมื่อสิ้นสุดกะของคุณ:

1. คลิก **ปิดกะ** บนแดชบอร์ดลิ้นชักเงินสดของคุณ
2. นับเงินสดจริงของคุณ
3. ป้อนจำนวน **เงินสดจริง**
4. ระบบจะแสดง:
   - **คาดหวัง**: สิ่งที่ควรอยู่ในลิ้นชัก
   - **จริง**: สิ่งที่คุณนับได้
   - **ส่วนต่าง**: ความแตกต่าง (ขาดหรือเกิน)
5. ถ้ามีส่วนต่าง:
   - เพิ่มหมายเหตุอธิบายความแตกต่าง
   - ยอมรับส่วนต่าง
6. คลิก **ปิดเซสชัน**

### การจัดการส่วนต่าง

| ส่วนต่าง | สถานะ | การดำเนินการ |
|----------|--------|--------|
| ไม่มี (0) | ปิดแล้ว | เซสชันปิดตามปกติ |
| ขาด (-) | ปิดพร้อมส่วนต่าง | ผู้จัดการจะตรวจสอบ |
| เกิน (+) | ปิดพร้อมส่วนต่าง | ผู้จัดการจะตรวจสอบ |

**เคล็ดลับสำหรับการกระทบยอดที่แม่นยำ:**
- นับเงินสดอย่างระมัดระวัง สองครั้งถ้าจำเป็น
- ตรวจสอบธนบัตรหรือเหรียญที่ติดอยู่
- ตรวจสอบธุรกรรมถ้าส่วนต่างไม่คาดคิด
- รายงานความคลาดเคลื่อนทันที

## การแก้ไขปัญหา

### ข้อผิดพลาด "ไม่มีลิ้นชักเงินสดที่ใช้งาน"
คุณพยายามดำเนินการเช่าโดยไม่มีลิ้นชักเงินสดที่เปิดอยู่
- **วิธีแก้**: เปิดลิ้นชักเงินสดของคุณก่อน แล้วลองใหม่

### ไม่สามารถเปิดลิ้นชักเงินสด
คุณอาจมีเซสชันที่เปิดอยู่สำหรับร้านนั้นแล้ว
- **วิธีแก้**: ปิดเซสชันที่มีอยู่ของคุณก่อน

### ส่วนต่างเมื่อปิด
เงินสดที่นับได้ไม่ตรงกับที่คาดหวัง
- **วิธีแก้**:
  1. นับเงินสดใหม่
  2. ตรวจสอบธุรกรรมสำหรับข้อผิดพลาด
  3. ตรวจสอบการจ่ายเงินที่พลาดไป
  4. บันทึกส่วนต่างและปิด

## แนวทางปฏิบัติที่ดี

1. **เปิดลิ้นชักเงินสดก่อนเริ่มงานเสมอ** - อย่าดำเนินการเช่าโดยไม่มี
2. **ขอใบเสร็จสำหรับการจ่ายเงินทุกครั้ง** - เก็บให้เป็นระเบียบ
3. **ฝากเงินสดเป็นประจำ** - อย่าเก็บมากเกินไปในลิ้นชัก
4. **นับอย่างระมัดระวังเมื่อปิด** - ใช้เวลาของคุณ
5. **รายงานปัญหาทันที** - อย่ารอจนถึงสิ้นวัน
6. **จดบันทึก** - บันทึกสิ่งผิดปกติใดๆ
