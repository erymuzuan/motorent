# คู่มือออกแบบเทมเพลตเอกสาร

เครื่องมือออกแบบเทมเพลตเอกสารเป็นเครื่องมือภาพแบบลากและวางที่ช่วยให้ผู้ดูแลองค์กรปรับแต่งเอกสารที่สร้างโดย MotoRent เช่น สัญญาเช่า การยืนยันการจอง และใบเสร็จ

## ภาพรวม

MotoRent ใช้ระบบเลย์เอาต์ตามส่วน เทมเพลตแต่ละอันประกอบด้วยหลายส่วน (ส่วนหัว เนื้อหา ส่วนท้าย) ที่มี "บล็อก" แบบลากได้เช่น ข้อความ ตาราง และรูปภาพ บล็อกเหล่านี้สามารถเชื่อมโยงกับข้อมูลไดนามิกจากการเช่าและการจองของคุณ

## เริ่มต้นใช้งาน

1. ไปที่ **การตั้งค่า > เทมเพลตเอกสาร**
2. คลิก **สร้างเทมเพลตใหม่** หรือคลิกที่เทมเพลตที่มีอยู่เพื่อแก้ไข
3. เลือก **ประเภทเทมเพลต**:
   - **การยืนยันการจอง**: ส่งให้ลูกค้าเมื่อจองยานพาหนะ
   - **สัญญาเช่า**: สัญญาทางกฎหมายที่ลงนามเมื่อเช็คอิน
   - **ใบเสร็จ**: หลักฐานการชำระเงินทางการเงิน

## การใช้งานเครื่องมือออกแบบ

อินเทอร์เฟซของเครื่องมือออกแบบแบ่งออกเป็นสามพื้นที่หลัก:
1. **กล่องเครื่องมือ (ซ้าย)**: มีบล็อกที่ลากได้
2. **พื้นที่ออกแบบ (กลาง)**: ที่คุณจัดเรียงบล็อกของคุณ
3. **โปรแกรมแก้ไขคุณสมบัติ (ขวา)**: กำหนดค่าการตั้งค่าสำหรับบล็อกที่เลือก

### บล็อกที่ลากได้

| ประเภทบล็อก | คำอธิบาย |
|------------|-------------|
| **ข้อความ** | ข้อความคงที่หรือโทเค็นข้อมูลไดนามิก (เช่น `{{CustomerName}}`) |
| **รูปภาพ** | โลโก้ของคุณหรือรูปภาพแบรนด์อื่นๆ |
| **ตาราง** | ใช้สำหรับรายการแบบรายการเช่นรายละเอียดยานพาหนะหรือรายละเอียดการชำระเงิน |
| **เส้นแบ่ง** | เส้นแนวนอนเพื่อแบ่งส่วน |
| **ส่วนหัว/ส่วนท้าย** | บล็อกที่กำหนดค่าไว้ล่วงหน้าสำหรับส่วนหัวและส่วนท้ายเอกสารมาตรฐาน |

### การเชื่อมโยงข้อมูล (โทเค็น)

คุณสามารถแทรกข้อมูลไดนามิกในบล็อกข้อความโดยใช้วงเล็บปีกกาคู่

**โทเค็นทั่วไป:**
- `{{OrganizationName}}` - ชื่อธุรกิจของคุณ
- `{{ShopName}}` - สถานที่ร้านค้าเฉพาะ
- `{{CustomerName}}` - ชื่อเต็มของผู้เช่า
- `{{VehicleName}}` - ยี่ห้อและรุ่นของรถ
- `{{StartDate}}` / `{{EndDate}}` - ระยะเวลาเช่า
- `{{TotalAmount}}` - ค่าใช้จ่ายทั้งหมดของการเช่า

## แนวทางปฏิบัติที่ดี

1. **ทดสอบด้วยการแสดงตัวอย่าง**: ใช้ปุ่ม "แสดงตัวอย่าง" เสมอเพื่อดูว่าเทมเพลตของคุณดูเป็นอย่างไรกับข้อมูลตัวอย่างก่อนใช้งานจริง
2. **ตั้งเป็นค่าเริ่มต้น**: เมื่อคุณพอใจกับเทมเพลตแล้ว คลิก **ตั้งเป็นค่าเริ่มต้น** เพื่อให้ใช้อัตโนมัติสำหรับการเช่า/การจองใหม่ทั้งหมด
3. **การสร้างแบรนด์**: อัปโหลดโลโก้ความละเอียดสูง (PNG หรือ JPG) สำหรับคุณภาพการพิมพ์ที่ดีที่สุด
4. **การปฏิบัติตามกฎหมาย**: ตรวจสอบให้แน่ใจว่าสัญญาเช่าของคุณมีข้อกำหนดและเงื่อนไขที่จำเป็นทั้งหมดตามที่กฎหมายท้องถิ่นกำหนด

## สถานะเทมเพลต

- **ร่าง**: งานที่กำลังดำเนินการ ไม่ปรากฏให้พนักงานเห็น
- **อนุมัติแล้ว**: พร้อมใช้งาน สามารถเลือกด้วยตนเองเมื่อพิมพ์
- **ค่าเริ่มต้น**: เทมเพลตหลักที่ระบบใช้อัตโนมัติ

## การแก้ไขปัญหา

### โทเค็นไม่แสดงข้อมูล
ตรวจสอบให้แน่ใจว่าคุณใช้ชื่อโทเค็นที่ถูกต้อง (คำนึงถึงตัวพิมพ์ใหญ่-เล็ก) และโทเค็นนั้นใช้ได้สำหรับประเภทเทมเพลต (เช่น `{{BookingID}}` ใช้งานได้เฉพาะบนเทมเพลตการจอง)

### เลย์เอาต์ดูเสียบนมือถือ
เครื่องมือออกแบบปรับให้เหมาะสมสำหรับการใช้งานบนเดสก์ท็อป เพื่อประสบการณ์ที่ดีที่สุด ออกแบบเทมเพลตของคุณบนหน้าจอขนาดใหญ่
