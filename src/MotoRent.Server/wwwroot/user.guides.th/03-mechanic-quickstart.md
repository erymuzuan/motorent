# คู่มือเริ่มต้นใช้งาน MotoRent - ช่างซ่อมบำรุง

ยินดีต้อนรับสู่ MotoRent! คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งานในฐานะช่างซ่อมบำรุง

![การจัดการยานพาหนะ](images/05-fleet.webp)

## บทบาทของคุณ

ในฐานะช่างซ่อมบำรุง คุณมุ่งเน้นที่การบำรุงรักษายานพาหนะ:
- ดูรายการยานพาหนะ
- ติดตามสถานะยานพาหนะ
- จัดการอุปกรณ์เสริม
- ตรวจสอบความต้องการการบำรุงรักษา

## ภาพรวมการนำทาง

เมนูนำทางของคุณประกอบด้วย:

| เมนู | คำอธิบาย |
|------|-------------|
| **แดชบอร์ด** | ภาพรวมสถานะยานพาหนะ |
| **ยานพาหนะ** | ดูยานพาหนะและอุปกรณ์เสริมทั้งหมด |

> **หมายเหตุ:** การดำเนินงานเช่า การเงิน และการตั้งค่าจัดการโดยพนักงานและผู้จัดการ

## เริ่มต้นใช้งาน

### 1. ภาพรวมแดชบอร์ด

แดชบอร์ดของคุณแสดง:
- **การเช่าที่ใช้งานอยู่** - ยานพาหนะที่ให้เช่าอยู่ในปัจจุบัน (ดูอย่างเดียว)
- **ยานพาหนะที่พร้อมใช้งาน** - พร้อมให้เช่า
- **ครบกำหนดวันนี้** - การคืนที่คาดหวัง
- **รายได้วันนี้** - ข้อมูลรายได้ (ดูอย่างเดียว)

### 2. การจัดการยานพาหนะ

ไปที่ **ยานพาหนะ > ยานพาหนะทั้งหมด** เพื่อดูรายการ

#### การดูยานพาหนะ

หน้ายานพาหนะแสดง:
- **ข้อมูลยานพาหนะ** - ป้ายทะเบียน ยี่ห้อ รุ่น
- **ประเภท** - มอเตอร์ไซค์ รถยนต์ เจ็ตสกี เรือ รถตู้
- **ราคา** - อัตราต่อวันและเงินมัดจำ
- **สถานะ** - ความพร้อมใช้งานปัจจุบัน
- **สถานที่** - ยานพาหนะอยู่ที่ร้านไหน

#### สถานะยานพาหนะ

| สถานะ | ความหมาย |
|--------|---------|
| **พร้อมใช้งาน** | พร้อมให้เช่า |
| **เช่าอยู่** | อยู่กับลูกค้าในปัจจุบัน |
| **ซ่อมบำรุง** | กำลังได้รับการบริการ - ไม่พร้อมใช้งาน |

#### การกรองยานพาหนะ

ใช้ตัวกรองเพื่อค้นหายานพาหนะเฉพาะ:
- **ทั้งหมด** - แสดงยานพาหนะทั้งหมด
- **มอเตอร์ไซค์** - รถสองล้อเท่านั้น
- **รถยนต์** - รถยนต์และรถซีดาน
- **เจ็ตสกี** - ยานพาหนะทางน้ำ
- **เรือ** - เรือ
- **รถตู้** - รถตู้และยานพาหนะขนาดใหญ่

ตัวกรองเพิ่มเติม:
- **สถานะ** - กรองตามความพร้อมใช้งาน
- **ยี่ห้อ** - กรองตามผู้ผลิต
- **รวมรถ Pool** - แสดงยานพาหนะที่แชร์ข้ามร้าน

### 3. อุปกรณ์เสริม

ไปที่ **ยานพาหนะ > อุปกรณ์เสริม** เพื่อดูอุปกรณ์เสริมให้เช่า:
- หมวกกันน็อค
- ที่ยึดโทรศัพท์
- อุปกรณ์ GPS
- ที่นั่งเด็ก
- อุปกรณ์อื่นๆ

ตรวจสอบรายการและสภาพของอุปกรณ์เสริม

### ระหว่างวัน
- [ ] ตรวจสอบสถานะการบำรุงรักษา
- [ ] อัปเดตสภาพยานพาหนะ

### การติดตามการบำรุงรักษา

มองหายานพาหนะที่ต้องการความสนใจ:
- ยานพาหนะที่ถูกทำเครื่องหมายสำหรับการบำรุงรักษา
- ยานพาหนะที่เสียหายจากอุบัติเหตุ
- ช่วงเวลาบริการตามปกติ

รายงานให้ผู้จัดการของคุณทราบหาก:
- ยานพาหนะต้องการการซ่อมแซมเร่งด่วน
- ต้องการอะไหล่
- ยานพาหนะควรถูกนำออกจากการให้บริการ

## ทำความเข้าใจประเภทยานพาหนะ

### มอเตอร์ไซค์
- ประเภทการเช่าที่พบบ่อยที่สุด
- ตรวจสอบ: ยาง เบรค ไฟ กระจก
- ปัญหาทั่วไป: รอยขีดข่วน กระจกเสียหาย

### รถยนต์
- ความต้องการเงินมัดจำสูงกว่า
- ตรวจสอบ: สภาพตัวถัง ภายใน แอร์
- ปัญหาทั่วไป: รอยบุบ คราบภายใน

### เจ็ตสกี
- การเช่าตามช่วงเวลา (รายชั่วโมง)
- ตรวจสอบ: ตัวเรือ เครื่องยนต์ อุปกรณ์ความปลอดภัย
- ปัญหาทั่วไป: รอยขีดข่วนตัวเรือ การบำรุงรักษาเครื่องยนต์

### เรือ
- อาจต้องการกัปตัน/ไกด์
- ตรวจสอบ: ตัวเรือ เครื่องยนต์ อุปกรณ์ความปลอดภัย น้ำมัน
- ปัญหาทั่วไป: การสึกหรอจากทะเล การบริการเครื่องยนต์

### รถตู้
- ยานพาหนะที่ใหญ่ที่สุด
- ตรวจสอบ: ทุกรายการของรถยนต์รวมถึงพื้นที่บรรทุก
- ปัญหาทั่วไป: การสึกหรอสูงกว่า

## การรายงานปัญหา

หากคุณพบปัญหายานพาหนะ:

1. บันทึกป้ายทะเบียนและปัญหา
2. รายงานให้ผู้จัดการร้านของคุณ
3. พวกเขาจะ:
   - อัปเดตสถานะยานพาหนะ
   - สร้างบันทึกการบำรุงรักษา
   - กำหนดเวลาการซ่อม

## เคล็ดลับสู่ความสำเร็จ

1. **ละเอียดถี่ถ้วน** - ตรวจสอบยานพาหนะอย่างระมัดระวังหลังการคืน
2. **บันทึกทุกอย่าง** - ถ่ายรูปความเสียหาย
3. **จัดระเบียบ** - ใช้ตัวกรองเพื่อติดตามงานของคุณ
4. **สื่อสาร** - รายงานปัญหาให้ผู้บริหารทราบทันที
