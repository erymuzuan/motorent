# คู่มือการคิดค่าเสื่อมราคาทรัพย์สิน MotoRent - การจัดการการเงิน

ยินดีต้อนรับสู่โมดูลค่าเสื่อมราคาทรัพย์สิน! คู่มือนี้ช่วยผู้ดูแลองค์กรและผู้จัดการร้านติดตามมูลค่ายานพาหนะ ค่าเสื่อมราคา ค่าใช้จ่าย และความสามารถในการทำกำไร

## ภาพรวม

ฟีเจอร์ค่าเสื่อมราคาทรัพย์สินมีให้:
- **การติดตามทรัพย์สิน** - เชื่อมโยงบันทึกการเงินกับยานพาหนะ
- **การคำนวณค่าเสื่อมราคา** - หลายวิธีในการติดตามการลดลงของมูลค่า
- **การจัดการค่าใช้จ่าย** - จัดหมวดหมู่และติดตามค่าใช้จ่ายยานพาหนะทั้งหมด
- **การติดตามเงินกู้** - การจัดการทางการเงินและตารางผ่อนชำระเต็มรูปแบบ
- **รายงานความสามารถในการทำกำไร** - การวิเคราะห์ ROI ต่อยานพาหนะ

## ใครควรใช้งาน

| บทบาท | ระดับการเข้าถึง |
|------|--------------|
| **OrgAdmin** | สิทธิ์เข้าถึงเต็มรูปแบบสำหรับฟีเจอร์ทรัพย์สินทั้งหมด |
| **ShopManager** | ดูและจัดการทรัพย์สินสำหรับร้านของตน |

## การนำทาง

เข้าถึงฟีเจอร์ทรัพย์สินจากเมนู **การเงิน**:

| รายการเมนู | คำอธิบาย |
|-----------|-------------|
| **การเงิน > แดชบอร์ดทรัพย์สิน** | ศูนย์บัญชาการทางการเงินพร้อม KPI กราฟ และการแจ้งเตือน |
| **การเงิน > ทรัพย์สิน** | ดูทรัพย์สินที่ติดตามทั้งหมดพร้อมสรุป |
| **การเงิน > รายละเอียดทรัพย์สิน** | ดูประวัติค่าเสื่อมราคาและกราฟมูลค่าสำหรับทรัพย์สินแต่ละรายการ |
| **การเงิน > รายงานค่าเสื่อมราคา** | สรุปค่าเสื่อมราคารายเดือนและรายปี |
| **การเงิน > ค่าเสื่อมราคา** | บันทึกและจัดการรายการค่าเสื่อมราคา |
| **การเงิน > ค่าใช้จ่าย** | ติดตามค่าใช้จ่ายดำเนินงานตามหมวดหมู่ |
| **การเงิน > เงินกู้** | จัดการสินเชื่อยานพาหนะ |
| **การเงิน > รายงาน > ความสามารถในการทำกำไร** | การวิเคราะห์ ROI ยานพาหนะ |

## แดชบอร์ดทรัพย์สิน - ศูนย์บัญชาการทางการเงิน

แดชบอร์ดทรัพย์สิน (`/finance/asset-dashboard`) ให้มุมมองที่ครอบคลุมสุขภาพทางการเงินของยานพาหนะของคุณในมุมมองเดียว

### การ์ดสรุป KPI

| การ์ด | แสดงอะไร |
|------|---------------|
| **ลงทุนทั้งหมด** | ต้นทุนการจัดหาทั้งหมดของยานพาหนะที่ติดตามทั้งหมด |
| **มูลค่าตามบัญชี** | มูลค่าปัจจุบันหลังหักค่าเสื่อมราคา (พร้อม % ของเดิม) |
| **ค่าเสื่อมราคาสะสม** | ค่าเสื่อมราคาทั้งหมดที่บันทึกพร้อมแถบความคืบหน้า |
| **กำไร/ขาดทุนสุทธิ** | รายได้ลบค่าใช้จ่ายลบค่าเสื่อมราคา |
| **ROI ยานพาหนะ** | เปอร์เซ็นต์ผลตอบแทนจากการลงทุนสำหรับยานพาหนะทั้งหมด |

### กราฟ

- **แนวโน้มค่าเสื่อมราคา** - กราฟ 12 เดือนแสดงการลดลงของมูลค่าตามบัญชีและค่าเสื่อมราคารายเดือน
- **รายละเอียดค่าใช้จ่าย** - กราฟโดนัทแสดงค่าใช้จ่ายตามหมวดหมู่ (การบำรุงรักษา การเงิน ประกัน ฯลฯ)

### การติดตามประสิทธิภาพ

- **ยานพาหนะที่ทำกำไรสูงสุด** - ยานพาหนะที่มี ROI สูงสุด จัดอันดับด้วยตัวบ่งชี้ทอง/เงิน/ทองแดง
- **ต้องการความสนใจ** - การแจ้งเตือนสำหรับ:
  - ทรัพย์สินที่ต้องคิดค่าเสื่อมราคารายเดือน
  - การชำระเงินกู้ที่ค้างชำระ
  - การชำระเงินที่กำลังจะมาถึง (7 วันข้างหน้า)
  - ยานพาหนะที่ทำกำไรต่ำ (ROI ติดลบ)

## เริ่มต้นใช้งาน

### 1. การสร้างบันทึกทรัพย์สิน

เพื่อติดตามค่าเสื่อมราคาสำหรับยานพาหนะ:

1. ไปที่ **การเงิน > ทรัพย์สิน**
2. คลิก **"+ เพิ่มทรัพย์สิน"**
3. กรอกรายละเอียดทรัพย์สิน:
   - **ยานพาหนะ** - เลือกยานพาหนะที่ต้องการติดตาม
   - **วันที่จัดหา** - เมื่อซื้อยานพาหนะ
   - **ต้นทุนการจัดหา** - ราคาซื้อ (THB)
   - **ชื่อผู้ขาย** - ตัวแทนจำหน่ายหรือผู้ขาย
   - **เลขที่อ้างอิง** - หมายเลขใบแจ้งหนี้หรือใบเสร็จ

4. กำหนดค่าการตั้งค่าค่าเสื่อมราคา:
   - **วิธี** - เลือกวิธีคำนวณค่าเสื่อมราคา
   - **อายุการใช้งาน** - อายุที่คาดหวังเป็นเดือน (ค่าเริ่มต้น: 60)
   - **มูลค่าซาก** - มูลค่าที่คาดหวังเมื่อสิ้นสุดอายุการใช้งาน

5. คลิก **บันทึก**

### 2. คำอธิบายวิธีการค่าเสื่อมราคา

| วิธี | วิธีการทำงาน | เหมาะสำหรับ |
|--------|--------------|----------|
| **Day Out of Door** | ค่าเสื่อมราคาทันทีเป็น % เมื่อเช่าครั้งแรก | มอเตอร์ไซค์ที่สูญเสียมูลค่าเมื่อใช้ครั้งแรก |
| **Straight Line** | จำนวนเท่ากันทุกเดือน | ยานพาหนะมาตรฐานที่มีการลดลงคาดการณ์ได้ |
| **Declining Balance** | % ของมูลค่าตามบัญชีปัจจุบัน | ค่าเสื่อมราคาสูงในช่วงแรก ช้าลงทีหลัง |
| **Custom** | กำหนดตารางรายเดือนเอง | สถานการณ์พิเศษที่ต้องการควบคุมด้วยตนเอง |
| **Hybrid** | Day Out of Door + Straight Line | รวมการลดลงครั้งแรกแล้วค่อยๆ ลดลงสม่ำเสมอ |

### 3. การบันทึกค่าเสื่อมราคา

ค่าเสื่อมราคาสามารถบันทึกอัตโนมัติหรือด้วยตนเอง:

#### การบันทึกอัตโนมัติ
1. ไปที่ **การเงิน > ค่าเสื่อมราคา**
2. คลิก **"คิดค่าเสื่อมราคารายเดือน"**
3. เลือกช่วงเวลา (เดือน/ปี)
4. ตรวจสอบจำนวนที่คำนวณ
5. คลิก **ยืนยัน**

#### การแก้ไขด้วยตนเอง
เมื่อคุณต้องการปรับจำนวนที่คำนวณ:

1. ไปที่ **การเงิน > ทรัพย์สิน > [ยานพาหนะ]**
2. คลิก **"บันทึกค่าเสื่อมราคา"**
3. ป้อนวันที่ของช่วงเวลา
4. เลือก **"แก้ไขจำนวนที่คำนวณ"**
5. ป้อนจำนวนที่กำหนดเอง
6. ให้เหตุผลสำหรับการแก้ไข
7. คลิก **บันทึก**

## การติดตามค่าใช้จ่าย

### หมวดหมู่ค่าใช้จ่าย

| หมวดหมู่ | ตัวอย่าง |
|----------|----------|
| **การบำรุงรักษา** | เปลี่ยนน้ำมัน เปลี่ยนยาง ซ่อมแซม |
| **ประกันภัย** | ค่าเบี้ยประกันรายปี |
| **การเงิน** | ดอกเบี้ยเงินกู้ |
| **อุบัติเหตุ** | ค่าซ่อมจากเหตุการณ์ |
| **การจดทะเบียน** | ค่าจดทะเบียนรายปี ภาษี |
| **วัสดุสิ้นเปลือง** | น้ำมันสำหรับการย้าย อุปกรณ์ทำความสะอาด |

### การบันทึกค่าใช้จ่าย

1. ไปที่ **การเงิน > ค่าใช้จ่าย**
2. คลิก **"+ เพิ่มค่าใช้จ่าย"**
3. เลือก **ทรัพย์สิน** (ยานพาหนะ)
4. เลือก **หมวดหมู่**
5. ป้อน **จำนวน** และ **วันที่**
6. เพิ่ม **ผู้ขาย** และ **เลขที่อ้างอิง** ถ้ามี
7. ทำเครื่องหมายถ้า **หักภาษีได้**
8. คลิก **บันทึก**

## การจัดการเงินกู้

### การตั้งค่าเงินกู้

เมื่อคุณสินเชื่อซื้อยานพาหนะ:

1. ไปที่ **การเงิน > เงินกู้**
2. คลิก **"+ เพิ่มเงินกู้"**
3. เลือก **ทรัพย์สิน**
4. ป้อนรายละเอียดเงินกู้:
   - **ชื่อผู้ให้กู้** - ธนาคารหรือบริษัทการเงิน
   - **เงินต้น** - จำนวนเงินกู้ (THB)
   - **อัตราดอกเบี้ยรายปี** - เช่น 8% = 0.08
   - **ระยะเวลา** - ความยาวเป็นเดือน
   - **เงินดาวน์** - การชำระเงินเริ่มต้น
   - **วันที่เริ่มต้น** - เมื่อเงินกู้เริ่มต้น

5. ระบบจะคำนวณ:
   - จำนวนเงินชำระรายเดือน
   - ตารางผ่อนชำระเต็มรูปแบบ

6. คลิก **บันทึก**

### การบันทึกการชำระเงินกู้

1. ไปที่ **การเงิน > เงินกู้ > [เงินกู้]**
2. ดูแท็บ **ตารางการชำระเงิน**
3. ค้นหาการชำระเงินที่ครบกำหนด
4. คลิก **"บันทึกการชำระเงิน"**
5. ป้อนวันที่ชำระเงินและเลขที่อ้างอิง
6. คลิก **ยืนยัน**

## การวิเคราะห์ความสามารถในการทำกำไร

### รายงาน ROI ยานพาหนะ

ไปที่ **การเงิน > รายงาน > ความสามารถในการทำกำไร** เพื่อดู:

- **ROI %** - ผลตอบแทนจากการลงทุนต่อยานพาหนะ
- **รายได้** - รายได้จากการเช่าทั้งหมด
- **ค่าใช้จ่าย** - ค่าใช้จ่ายทั้งหมดตามหมวดหมู่
- **กำไร/ขาดทุนสุทธิ** - บรรทัดสุดท้ายต่อยานพาหนะ

### ตัวชี้วัดสำคัญ

| ตัวชี้วัด | สูตร |
|--------|---------|
| **ฐานที่เสื่อมราคาได้** | ต้นทุนการจัดหา - มูลค่าซาก |
| **มูลค่าตามบัญชีปัจจุบัน** | ต้นทุนการจัดหา - ค่าเสื่อมราคาสะสม |
| **กำไร/ขาดทุนสุทธิ** | รายได้ - ค่าใช้จ่าย - ค่าเสื่อมราคา |
| **ROI %** | กำไรสุทธิ / ต้นทุนการจัดหา × 100 |

## การจำหน่ายทรัพย์สิน

### การขายยานพาหนะ

เมื่อคุณขายยานพาหนะที่ติดตาม:

1. ไปที่ **การเงิน > ทรัพย์สิน > [ยานพาหนะ]**
2. คลิก **"จำหน่ายทรัพย์สิน"**
3. ป้อน:
   - **วันที่จำหน่าย**
   - **จำนวนเงินขาย**
4. ระบบคำนวณ **กำไร/ขาดทุน**:
   - จำนวนเงินขาย > มูลค่าตามบัญชี = กำไร
   - จำนวนเงินขาย < มูลค่าตามบัญชี = ขาดทุน
5. คลิก **ยืนยัน**

### การตัดบัญชียานพาหนะ

สำหรับยานพาหนะที่ไม่มีมูลค่าขายต่อ (การโจรกรรม ความเสียหายทั้งหมด):

1. ไปที่ **การเงิน > ทรัพย์สิน > [ยานพาหนะ]**
2. คลิก **"ตัดบัญชี"**
3. ป้อน **เหตุผล** สำหรับการตัดบัญชี
4. ระบบบันทึกมูลค่าตามบัญชีทั้งหมดเป็นขาดทุน
5. คลิก **ยืนยัน**
