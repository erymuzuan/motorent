# คู่มือการจัดการตัวแทน

การจัดการตัวแทนช่วยให้คุณติดตามการจองและการเช่าที่นำเข้ามาโดยพันธมิตรบุคคลที่สาม เช่น โรงแรม บริษัททัวร์ หรือตัวแทนท้องถิ่น คุณสามารถจัดการค่าคอมมิชชันและติดตามผลงานของตัวแทนแต่ละรายได้

## การลงทะเบียนตัวแทน

1. ไปที่ **ตัวแทน** ในแถบด้านข้าง
2. คลิก **เพิ่มตัวแทน**
3. กรอกรายละเอียดของตัวแทน:
   - **ชื่อ**: ชื่อบุคคลหรือชื่อธุรกิจ
   - **ข้อมูลติดต่อ**: โทรศัพท์และอีเมล
   - **ประเภทคอมมิชชัน**: เปอร์เซ็นต์หรือจำนวนเงินคงที่ต่อการจอง
   - **อัตราเริ่มต้น**: ค่าคอมมิชชันมาตรฐานที่คุณจ่ายให้ตัวแทนนี้
4. คลิก **บันทึก**

## การเชื่อมโยงการจองกับตัวแทน

เมื่อลูกค้ามาจากตัวแทน:

### ระหว่างการจอง
เลือก **ตัวแทน** จากเมนูดรอปดาวน์ในวิซาร์ดการจอง ระบบจะคำนวณค่าคอมมิชชันที่คาดหวังโดยอัตโนมัติ

### ระหว่างเช็คอิน (ลูกค้า Walk-in)
หากลูกค้าเดินเข้ามาและกล่าวถึงตัวแทน:
1. เริ่มกระบวนการ **เช็คอิน**
2. ในฟิลด์ **ตัวแทน** เลือกตัวแทนที่แนะนำลูกค้ามา

## การติดตามคอมมิชชัน

### สถานะคอมมิชชัน
- **รอดำเนินการ**: การจองยังใช้งานอยู่แต่ยังไม่ได้จ่ายคอมมิชชัน
- **ได้รับแล้ว**: การเช่าเสร็จสมบูรณ์ ถึงกำหนดจ่ายคอมมิชชัน
- **จ่ายแล้ว**: คอมมิชชันได้รับการบันทึกว่าจ่ายแล้วในลิ้นชักเก็บเงิน

### การประมวลผลการจ่ายเงิน
เพื่อจ่ายเงินให้ตัวแทน:
1. ไปที่หน้า **ลิ้นชักเก็บเงิน**
2. คลิก **บันทึกการจ่ายเงิน**
3. เลือก **คอมมิชชันตัวแทน**
4. เลือกตัวแทนและการเช่า/การจองที่เฉพาะเจาะจง
5. กรอกจำนวนเงินและยืนยัน

## รายงานผลงานตัวแทน

ไปที่ **การเงิน > รายงาน > ผลงานตัวแทน** เพื่อดู:
- จำนวนการจองทั้งหมดต่อตัวแทน
- รายได้รวมที่สร้างขึ้น
- คอมมิชชันที่จ่ายแล้วเทียบกับที่ค้างจ่าย
